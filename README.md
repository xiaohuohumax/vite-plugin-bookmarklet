# vite-plugin-bookmarklet

这是一个 vite 插件项目，可以将项目打包构建成一个可以直接在浏览器书签栏中运行的 bookmarklet。

## ♻️ 提交流程

1. 编写代码
2. [可选] 修改版本号 `pnpm changeset`
3. 拉取最新代码 `git pull`
4. 本地提交 `pnpm commit`
5. 推送代码到远程仓库
